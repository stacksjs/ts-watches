All contributors are listed on [GitHub](https://github.com/stacksjs/ts-watches/graphs/contributors).

## Join Us

Interested in contributing? We'd love to have you! Here are some ways to get involved:

- **Report bugs** - Found an issue? Open a [bug report](https://github.com/stacksjs/ts-watches/issues/new)
- **Request features** - Have an idea? Start a [discussion](https://github.com/stacksjs/ts-watches/discussions)
- **Submit PRs** - Fix bugs or add features
- **Improve docs** - Documentation improvements are always welcome
- **Help others** - Answer questions on Discord or GitHub

[Join the Stacks Discord](https://discord.gg/stacksjs) to chat with the team and other users.
